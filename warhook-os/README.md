# WARHOOK OS

Единое PWA-приложение вселенной художника **WARHOOK LEO CROC** (Роман Крюков): генератор арта, витрина серии, паспорта картин. Ставится на телефон как нативное приложение, работает офлайн.

**Онлайн:** https://trainingsw75-cloud.github.io/warhook-os/

## Установка на телефон
- **iPhone (Safari):** открой ссылку → «Поделиться» → «На экран Домой».
- **Android (Chrome):** открой ссылку → меню (⋮) → «Установить приложение» / «Добавить на главный экран».

Появится иконка **WARHOOK** рядом с остальными приложениями.

## Как обновлять (без кода)
Всё содержимое — в одном файле **`data.json`**.

Поменять статус картины — открой `data.json`, найди нужную работу и поменяй `status`:
- `"wip"` — 🎨 В процессе
- `"ready"` — ✅ Готова к продаже
- `"sold"` — 🔴 Продана
- `"collection"` — 🏛 В коллекции

Сохрани, закоммить — приложение у всех обновится само (service worker подтянет свежий `data.json`).

## Файлы
| Файл | Что делает |
|---|---|
| `index.html` | Приложение: экраны, навигация, встроенный генератор |
| `manifest.json` | Манифест PWA (иконка, имя, standalone) |
| `sw.js` | Service Worker — кэш и офлайн-режим |
| `data.json` | Единый источник правды (серия, статусы) |
| `icon-192.png` / `icon-512.png` | Иконки приложения |
